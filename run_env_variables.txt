run in terminal where main.py is:
--------------------------------

$env:FLASK_APP="main.py"
$env:FLASK_RUN_PORT="5001"
$env:FLASK_DEBUG="True"   
$env:FLASK_ENV="development"

run flask server
-----------------
flask run


run seed files
--------------
NOTE: This way, Python automatically recognizes extensions.py as part of your package

python -m seed.create_tables

python -m seed.create_subjects_classes
python -m seed.create_students_school_records
python -m seed.create_teacher_school_records

python -m seed.associate_teachers_to_subjects


python -m seed.associate_students_to_attendances
python -m seed.associate_students_to_grades
python -m seed.create_assignments_associate_fileupload
python -m seed.associate_student_to_assignment_submission
